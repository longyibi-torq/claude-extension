# Claude Code Extension Guide

An interactive reference site for the Claude Code extension ecosystem — covering Commands, MCP/Tools, Skills, Hooks, and Plugins.

## Live Site

Visit the site at: **[your-username.github.io/claude-extension](https://your-username.github.io/claude-extension)**

## Pages

- **Ecosystem Diagram** — Interactive visual map showing how Commands, MCP, Skills, Hooks, and Plugins fit together, ordered by learning sequence
- **Extension Concepts** — Detailed reference with summary tables, installation methods, popular examples, and hook lifecycle documentation

## Setup for GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings > Pages**
3. Set Source to **Deploy from a branch**
4. Set Branch to **main** and folder to **/docs**
5. Save — your site will be live in a few minutes

## Project Structure

```
docs/                       # Published site (GitHub Pages serves from here)
├── index.html              # Landing page
├── concepts.html           # Extension concepts reference
└── ecosystem-diagram.html  # Interactive ecosystem diagram
```
